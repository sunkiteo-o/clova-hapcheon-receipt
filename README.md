# 회계를 부탁해 😎

서울 중앙 교회 그루터기 2026 하기봉사 회계 관리 웹앱입니다.  <br/>
영수증을 촬영하면 연결된 하기봉사 장부·증빙 Google Sheets에 자동 기록합니다. <br/>
영수증 이미지는 Vercel Blob에 저장하며 관리합니다. 

  <img width="300" alt="IMG_9408" src="https://github.com/user-attachments/assets/71c07a56-b0b6-43f0-9073-b00ba5509d4c" />


## User Flow

[일반/ 취사] 구분 선택 → 영수증 촬영 → 내용 입력 → 저장 → 장부 시트(지출 기록) + 증빙 시트(이미지 수식) 동시 저장 완료

## 실행

```bash
pnpm install
pnpm dev
```
