# 회계를 부탁해 😎

영수증을 촬영하면 CLOVA OCR로 날짜·상호·금액을 자동 추출하여 Google Sheets에 저장하는 모바일 웹 앱입니다.

## 흐름

1. 팀 선택 → 영수증 사진 촬영 → 분석
2. OCR 결과 확인 및 수동 수정
3. 항목 입력 후 저장 → 시트에 자동 기록


## 실행

```bash
pnpm install
pnpm dev
```

## Google Sheets 시트 구조

탭별로 아래 헤더가 필요합니다.

| 날짜 | 팀 | 상호 | 항목 | 금액 | 상태 | 입력시간 |
|------|-----|------|------|------|------|----------|

- 취사팀 → `취사` 탭 (또는 `TAB_CHWISA`)
- 나머지 팀 → `통합` 탭 (또는 `TAB_COMMON`)

## 시트 연결 테스트

```bash
pnpm test:sheet
```
